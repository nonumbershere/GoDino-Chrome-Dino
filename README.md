# Installation
1) Go to [Chrome Dino](chrome://dino)
2) Press F12 and go to the Console tab
3) Paste the code from [Here](https://github.com/nonumbershere/GoDino-Chrome-Dino/blob/main/godino.min.js)
5) Done!
